My Projects!

Here are some of the projects I've built, applying the skills and technologies mentioned above. Each project showcases my ability to create clean, functional, and visually appealing applications, from front-end designs to full-stack implementations.

AthletIQ

AthletIQ is a dynamic gym workout platform built with React, Redux, HTML, CSS, JavaScript, Express, PostgreSQL, and Sequelize power the back end. It allows users to create custom workout plans that others can view, try out, and review. The intuitive design makes it easy to navigate, and the expandable left sidebar provides plenty of room for future enhancements and features.

AirBnB Clone

This is a full-stack Airbnb clone built using React, Redux, HTML, CSS, JavaScript, PostgreSQL, and Sequelize. It allows users to list properties, make bookings, and leave reviews in a smooth and intuitive interface. The design focuses on user-friendly navigation, and there’s plenty of potential for expanding features to enhance the booking experience even further.
